# Jekyll Blog

这是一个使用Jekyll构建的个人博客项目，旨在分享我的旅行经历和技术知识。

## 项目结构

- **_config.yml**: Jekyll的配置文件，包含网站的基本设置。
- **_posts/**: 存放博客文章的目录。
  - **2023-10-01-nanjing-trip.md**: 记录了我去南京旅游的经历，附有风景图。
  - **2023-10-01-jekyll-tutorial.md**: 分享了Jekyll的使用技巧和技术细节。
- **_includes/**: 包含网站的头部、底部和侧边栏文件。
- **_layouts/**: 定义网站整体结构的布局文件。
- **_site/**: Jekyll生成的静态网站文件夹。
- **assets/**: 存放样式表、JavaScript文件和图片的目录。
  - **css/style.css**: 自定义样式表。
  - **js/script.js**: 自定义JavaScript文件。
  - **images/**: 存放头像和风景图片。
- **index.html**: 网站的主页，包含欢迎信息和最新文章的链接。

## 使用指南

1. 确保已安装Jekyll。
2. 克隆此项目到本地。
3. 在项目根目录下运行 `jekyll serve` 启动本地服务器。
4. 打开浏览器访问 `http://localhost:4000` 查看博客。
